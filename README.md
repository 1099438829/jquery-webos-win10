jquery-webos-win10介绍
========================

jquery-webos-win10，是一个模拟WIN10系统桌面的示例程序。

## 使用组件
#### CSS组件

```{bash}				
1.基础CSS Metro-UI-CSS								
```
#### jquery组件

```{bash}
1.右键菜单 jquery.contextMenu.js			  	
2.桌面布局 jquery.gridster.js					
3.弹出层 jquery.jspanel.js					
4.模块化加载 require.js							
5.滚动条扩展 jquery.mCustomScrollbar.js			
6.MVVM框架 vue.js								
```
